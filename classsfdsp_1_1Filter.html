Detailed Description

Base class for filters.

Differential equations where taken from here: https://ccrma.stanford.edu/~jos/filters/Elementary_Filter_Sections.html Implementation for a lot of these functions were also taken from the STK (Synthesis ToolKit) library: https://github.com/thestk/stk
