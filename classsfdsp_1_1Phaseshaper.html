void sfdsp::Phaseshaper::SetWaveform ( float  wave)

Sets the waveform.

Parameters
wavefloating point value between Waveform::VARIABLE_SLOPE and Waveform::NUM_WAVES
Note
The fractional part of the value is used to interpolate between waveforms. For example, a value of 1.5 will interpolate between Waveform::SUPERSAW and Waveform::WAVESLICE equally.
