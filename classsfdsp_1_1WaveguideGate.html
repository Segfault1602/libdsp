Detailed Description

Implements a "reflection" point in a waveguide.

This class is used to implement a "reflection" point in a waveguide, for example, a finger on a string. While it can look like a scattering junction, the math is slightly different. The implementation is based on the procedure described in "Discrete-Time Modeling of Acoustic Tubes Using Fractional Delay Filters" by Vesa Välimäki
